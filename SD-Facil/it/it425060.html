Como fazer em casos de erro ‘ORA – 00054: Resource busy and acquire With NO WAIT especified’.

OBJETIVO:

Orientar o analista em como atuar nos chamados referentes ao erro “CRM-0181”, ocasionados no sistema NETSMS, quando o usuário tenta cancelar uma solicitação que existe OS em andamento pela rede interna.

APLICAÇÃO:

Essa IT possuí aplicabilidade nos seguintes grupos:
COTI

REFERÊNCIAS:

N/A

DESCRIÇÃO GERAL DO PROCEDIMENTO:

Usuário relata erro abaixo para realizar qualquer procedimento no sistema NETSMS. Sempre que ocorrer esse tipo de erro, o chamado deverá ser encaminhado à equipe COTI, pois houve algum lock na base de dados.

O colaborador deverá abrir um chamado do tipo incidente, anexando print do erro e detalhando a quantidade de usuários afetados. Este chamado deverá ser tratado como INCIDENTE CRITICO.


Classificação:Incidente / Crítico
Grupo Solucionador:COTI
Categoria:SOFTWARE.SISTEMAS DE RELACIONAMENTO.NET-SMS.PERFORMANCE
