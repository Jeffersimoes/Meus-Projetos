Não é possível cancelar solicitações com OS executada


OBJETIVO:

Este documento visa orientar o analista do N1 para casos onde ocorre erro ao realizar o cancelamento de uma OS, o sistema apresenta
erro informando ”alguma OS já foi baixada”.


APLICAÇÃO:

Essa IT possuí aplicabilidade nos seguintes grupo:

  • Service Desk N1

REFERÊNCIAS:

N/A


DESCRIÇÃO GERAL DO PROCEDIMENTO

Quando o usuário entra em contato informando que não consegue cancelar uma solicitação, devemos identificar o erro que aparece ao
realizar esta ação.

Caso o erro seja o que se encontra abaixo, devemos encerrar o chamado em N1, conforme instrução do item solução.



SOLUÇÃO:

Encerrar o chamado conforme categoria abaixo:


Tipo de chamado: Incidente:

Categoria A1: SISTEMAS.NETSMS.SOLICITACOES.BAIXA (OS / OC JÁ BAIXADAS)

Grupo Solucionador: Service Desk N1


SUGESTÃO DE RESPOSTA:

Prezado usuário(a),

A mensagem informa pelo sistema não caracteriza uma falha e sim uma informação.

Não será possível realizar o cancelamento da solicitação porque a OS que está atrelada a solicitação desejada, se encontra executada.

Atenciosamente,
Service Desk Claro Brasil
