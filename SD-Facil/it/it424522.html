Acesso a Permissões de Acesso (PA's)

IT:424522

OBJETIVO:

Esse procedimento tem o objetivo, orientar o analista em como proceder quando o usuário solicita acesso ao sistema de Permissão de Acesso.


APLICAÇÃO:

• Service Desk N1

• Web-Team


REFERÊNCIAS:

N/A


DESCRIÇÃO GERAL DO PROCEDIMENTO

Quando o usuário solicita acesso ao sistema NET Online, que é o sistema de permissão de acesso, devemos verificar se usuário possui login N, funcionário NET ou login iniciando com a letra T, terceirizada.


Caso o usuário possua login iniciando com a letra N, devemos abrir um chamado de solicitação preenchendo os dados abaixo e direcionar conforme instrução no item solução:

Dados para solicitação de login N:

  • Nome:
  • Login:
  • Data de Nascimento:
  • CPF:
  • RG:
  • Centro de Custo:
  • Departamento:
  • Função:
  • Cidade:
  • Gestor:

Caso o usuário possua login iniciando com a letra T, devemos verificar se existe uma PA aprovada para que possamos direcionar o chamado, conforme instrução no item solução. Se o usuário não tiver uma PA aprovada, o chamado deverá ser encerrado solicitando que abra uma PA conforme instrução no item solução e sugestão de resposta.


Após a aprovação da PA o usuário poderá abrir o chamado do tipo ”SOLICITAÇÃO” na ferramenta CA Service Desk. Manager, seguindo o formulário abaixo.


  • Nome:
  • Login:
  • Data de Nascimento:
  • CPF:
  • RG:
  • Centro de Custo:
  • Departamento:
  • Função:
  • Cidade:
  • Gestor:

SOLUÇÃO:


Caso a PA tenha sido aprovada ou no usuário seja funcionário NET, login N, devemos direcionar o chamado conforme instrução abaixo.

Classificação: Solicitação - A3

Categoria:SOFTWARE.APLICACOES WEB.PA.ACESSO AS SOLICITACOES ONLINE

Grupo: WEB-TEAM

SUGESTÃO DE RESPOSTA:

Caso 1 – Transferência para WEB TEAM se login do usuário começar com a letra T.


Prezado,


Usuário solicita liberação de acesso para o sistema_____________

Após análise foi identificado que existe uma PA aprovada para liberação de acesso.

Favor prosseguir com a tratativa.


Atenciosamente,

Service Desk Claro Brasil



Caso 2 – Transferência para WEB TEAM se login do usuário começar com a letra N.


Prezado,


Usuário solicita liberação de acesso para o sistema_____________

Após análise foi identificado que login do usuário começa com a letra N. Nesse caso, conforme IT 424522 não há necessidade de PA aprovada.

Favor prosseguir com a tratativa.


Atenciosamente,

Service Desk Claro Brasil

Caso 3 – Colaborador abriu um chamado sem PA

Prezado Usuário,


Após análise identificamos que o chamado está fora do escopo.

Chamados requerendo acesso ao sistema é necessário possuir uma PA aprovada.

Por gentileza entrar em contato com seu supervisor direto solicitando abertura da PA e após aprovação abrir um novo chamado do tipo solicitação.


Atenciosamente,

Service Desk Claro Brasil



Caso 4 – Chamado aberto como incidente

Prezado Usuário,


Este chamado será encerrado por se encontrar fora de escopo.

Para chamado de solicitação de acessos é obrigatório abertura do chamado do tipo “solicitação” e com PA aprovada pelo superior direto.


Atenciosamente,

Service Desk Claro Brasil
