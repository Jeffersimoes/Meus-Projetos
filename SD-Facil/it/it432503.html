Funcionalidade Associar Terminal ao Ponto

OBJETIVO:

Orientar o analista o passo a passo para a associar terminal ao ponto para melhor orientar o usuário que desconheça o procedimento.


APLICAÇÃO:

Essa IT possuí aplicabilidade nos seguintes grupos:

  • SERVICE DESK N1

DESCRIÇÃO GERAL DO PROCEDIMENTO:

A primeira validação que o analista deverá fazer é verificar no HPBT o status do terminal poderá ter os seguintes status:

  • Inicializado
  • Não inicializado
  • Empreiteira
  • Perda e roubo

Para fazer a validação o analista deverá seguir os passos abaixo:

  • Acessar o HP Business Tool: Clique aqui

Procedimento de Consulta:

  • Acessar o HP Business Tool;
  • Selecionar ‘Consultar Ação’;
  • Selecionar a Cidade desejada;
  • Selecionar o ‘Grupo’ – “OSS”;
  • Selecionar o ‘Macro Processo’ – “OSS / Atlas”;
  • Selecionar a ‘Funcionalidade’ – “Informações do Terminal”;
  • Selecionar a ‘Ação’ – “CONSULTA DAS INFORMAÇÕES DE UM TERMINAL

É necessário o status do terminal esteja como inicializado para que o terminal seja associado ao contrato.

Após realizar essa validação o analista deverá instruir o colaborador abrir o Netsms. Acessar a tela através do menu: REDE INTERNA >> OS’s >> SUPERVISÃO DE REDE INTERNA >> ASSOCIAR TERMINAL AO PONTO


SOLUÇÃO:

Esse procedimento orienta passo a passo como associar um terminal ao ponto e não se aplica para erros que possam ocorrer.

Caso ocorra algum erro durante esse procedimento o analista deverá consultar outra IT.


SUGESTÃO DE RESPOSTA:

Personalize a resposta para o analista, sempre informando o motivo do encerramento e ou o motivo do direcionamento para determinada torre.


Caso 1 – Terminal Inicializado

Prezado usuário,

Após fazer análise no HPBT, foi verificado que o status do terminal encontra-se inicializado.
Favor seguir o procedimento abaixo para fazer a associação.
Abrir o Netsms acessar a tela através do menu: REDE INTERNA >> OS’s >> SUPERVISÃO DE REDE INTERNA >> ASSOCIAR TERMINAL AO PONTO. Após localizar o contrato deverá selecionar o terminal que deseja associar junto ao ponto e clicar em “Salvar”.

Atenciosamente,
Service Desk Claro Brasil


Caso 2 – Terminal Não Inicializado

Prezado usuário,

Para poder fazer a associação é necessário que o terminal esteja inicializado e conforme verificado o status do terminal não esta inicializado, não sendo possível fazer a associação.

Atenciosamente,
Service Desk Claro Brasil
