Como fazer Entrega/Devolução de terminais em baixa de OS

RESUMO:

PROBLEMA:

RESOLUÇÃO:

Para todos os casos onde o usuário solicita entrega / devolução de terminal ao ponto (Associar terminal ao ponto) e não informa erro, fazer a seguinte pergunta ao usuário:

Qual o status do contrato?

Desconexão Opção - o usuário deve utilizar a OS da solicitação de desconexão opção e efetuar devolução do terminal (utilizando a orientação padrão).

Desconexão Inadimplência com Retirada de Equipamento - o usuário deve utilizar a OS da solicitação de desconexão inadimplência e efetuar a devolução do terminal (utilizando a orientação padrão).

Desconectado Inadimplência (total) - o usuário deve utilizar a OS da solicitação de desconexão inadimplência e efetuar a devolução (utilizando a orientação padrão).

Reinstalação Assinatura - o usuário deve utilizar a OS da solicitação de reinstalação assinatura e efetuar a entrega do terminal (utilizando a orientação padrão).

Reinstalação Ponto Adicional - o usuário deve utilizar a OS da solicitação de reinstalação ponto adicional e efetuar a entrega do terminal (utilizando a orientação padrão).

Entrar no menu Rede Interna > OSs > Supervisão de Rede Interna > Supervisão de rede interna principal e informar a OS em que o terminal deveria ser entregue/devolvido e clicar no botão “Terminais”.

As operação Entrega e Devolução/Retirada deve ser feita com usuário Supervisor de Rede Interna.

OBS 1.: Caso o sistema emita erro ao utilizar o menu Supervisor de Rede interna o mesmo emitira uma mensagem que o usuário não possui perfil.

Orientar o usuário para solicitar o perfil necessário ou solicitar há algum usuário que possua tal perfil, realize o processo.

Para este caso efetuar abertura de um chamado com os dados abaixo;

OBS 2 .: AS TELAS ABAIXO SÃO APENAS UM EXEMPLO DO PASSO A PASSO A SER SEGUIDO PARA OS CASOS EM QUE O USUÁRIO DESEJA EFETUAR A DEVOLUÇÃO/RETIRADA DE UM TERMINAL.

- DEVOLUÇÃO/RETIRADA

Na aba técnico é possível ver as OSs de retirada que estão abertas.

- Clicar em Rede Interna – OS’s – Baixa de OS’s.

- Informar o número da OS (no caso do exemplo uma OS de retirada, desconexão) o contrato e clicar em OK.

- Será preenchida a tela abaixo automaticamente, escolher a AÇÃO, no caso do exemplo como é uma retirada escolher DEVOLUÇÃO (DECODER DIGITAL, SMART....).

- Abaixo informar o número do equipamento que vai ser devolvido, escolher o estado (ex: suspeito) que ele deve ser devolvido e clicar no botão SALVAR.

Para este caso utilizar os dados abaixo para abertura do chamado. O processo é valido tanto para baixa de OS via COP quando Baixa de OS via Supervisão de rede interna.
(Sempre questionar o usuário para efetuar abertura de chamado com a categoria correta)
