CRM-0104: Não é permitido que se realize esta movimentação

OBJETIVO:

Esse procedimento tem o intuito de orientar o analista como deverá proceder quando o sistema apresentar erro CRM-0104, não permitindo realizar a movimentação de terminal.

APLICAÇÃO:

Essa IT possuí aplicabilidade nos seguintes grupos:

  • SERVICE DESK N1
  • Atlas

REFERÊNCIAS:

  • N/A

DESCRIÇÃO GERAL DO PROCEDIMENTO

Caso 1: Colaborador entra em contato reportando mensagem de erro ao tentar realizar ENTREGA/DEVOLUÇÃO de equipamento.



Caso 2: Sistema apresenta mensagem de erro ao tentar realizar a movimentação/baixa de OS’s.



SOLUÇÃO:

Nestes casos, deverá ser aberto chamado tipo incidente utilizando uma das categorias abaixo, dependendo do procedimento que o usuário deseja realizar.

  • SISTEMAS.NETSMS.TERMINAIS ENTREGA.TELA DE SUPERVISAO DE REDE INTERNA
  • SISTEMAS.NETSMS.TERMINAIS DEVOLUCAO.TELA DE SUPERVISAO DE REDE INTERNA
  • SISTEMAS.NETSMS.TERMINAIS ENTREGA.EM BAIXA DE OS

Para as categorias citadas o grupo solucionador é Atlas.

Lembrando que para cada categoria a aba propriedades solicita informações especificas, que deverá ser preenchida corretamente.


SUGESTÃO DE RESPOSTA:

Prezado(a),

Durante análise do caso, identificamos que ao realizar a tentativa de movimentação de terminal, ocorre o erro CRM-0104, conforme procedimento descrito na IT425064, estamos encaminhando o chamado para este grupo solucionador. Solicitamos apoio na tratativa do caso.

Atenciosamente,
