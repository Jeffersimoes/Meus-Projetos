Exceção / Bloqueio para Mudança de Pacote

OBJETIVO:

Esse procedimento tem o objetivo de verificar se o Login possui acesso para abertura de solicitação de mudança de pacote.


APLICAÇÃO:

Essa IT possuí aplicabilidade nos seguintes grupos:

  • Service Desk N1

REFERÊNCIAS:

N/A


DESCRIÇÃO GERAL DO PROCEDIMENTO

O usuário entra em contato reportando problema, produto indisponível ou não consegue localizar produto, nesse caso o analista deverá verificar se o usuário possui acesso para realizar a mudança de pacote.

Primeiro passo é acessar o HPBT seguindo o escopo abaixo.


  • Acessar o HP Business Tool: Clique aqui
  • Grupo : CRM
  • Macroprocesso : CRM / ACESSO / PERFIL
  • Funcionalidade : PERFIL / USUÁRIO
  • Ação : CONSULTAR EXCESSÃO PARA GERAR SOLICITAÇÃO

Se o usuário não possuir acesso o chamado deverá ser encerrado em N1 seguindo a sugestão de resposta 7.

Caso o usuário tenha acesso, o analista deverá verificar o erro e consultar na base outro procedimento pois esse procedimento tem o intuito somente para validação e não auxilia em caso de erros.


SOLUÇÃO:

O chamado deverá ser mantido em N1 independente do resultado da consulta a permissão do usuário.

Caso usuário tenha permissão esse procedimento não se aplica, pois, o objetivo desse procedimento é somente validar acessos, não orienta a fazer direcionamento ou transferência.


SUGESTÃO DE RESPOSTA:

Caso 1 – Usuário não possui permissão

Prezado usuário(a),

Verificado no sistema e o Login informando não possui acesso para realizar esse procedimento.
Favor caso seja necessário realizar esse procedimento é necessário entrar em contato com sua supervisão direta solicitando acesso.

Atenciosamente,
Service Desk Claro Brasil
